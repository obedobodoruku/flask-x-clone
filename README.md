A mini social media platform built with Flask, HTML and tailwindcss
---
# Features
- User registration and login
- Secure authentication with sessions
- Create, edite, and delete posts
- Reply to posts
- User-based permissions (only users can edit/delete their own posts and replies)
- Clean and simple UI
---
# Tech Stack
- Backend: Flask (Python)
- Frontend: HTML, CSS, Tailwindcss. Jinja2
- Database: SQLite
- Authentication: Flask Sessions
- Version Control: Git & GitHub
